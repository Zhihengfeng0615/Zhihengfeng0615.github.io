程老师分享的一些治学心得

发表于 2018-09-23 更新于 2020-08-14 阅读次数：
本文字数： 369 阅读时长 ≈ 1 分钟

好的习惯很重要；
国家情怀；
北大精神；
数学如果成为灵魂中的一部分，就能够创造；

  1. 程老师的学习习惯非常好，今天他提到了一个统一：历史、物理图像、数学公式三者的统一。每接触到一个新的东西，它一定存在历史，然后就是用物理图像和数学公式去描述它。
  2. 物理图像对应于粒子级别的运动模式。宏观物体的现象都应该对应的有粒子运动。粒子如何运动其实就是物理图像。
  3. 程老师对数学的理解和运用，令人望尘莫及。数学既能够对问题进行抽象，又能够发现解决问题的途径。
  4. 以后再给别人讲东西，先做prepare，比如，要向别人讲A-Z，我们应该补充A-B，B-C，……，Y-Z这些内容，一步一步地讲，这样别人就能听懂了，不能直接就丢给别人一个结论或者一个包裹。思考问题就像是爬山，一个台阶一个台阶的来，每个台阶都由我们传授给别人，最后我们想讲清楚的东西自然也就顺理成章被别人接受了。
