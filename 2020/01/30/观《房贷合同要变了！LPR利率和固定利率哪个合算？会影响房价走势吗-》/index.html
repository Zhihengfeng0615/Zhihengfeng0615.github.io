观《房贷合同要变了！LPR利率和固定利率哪个合算？会影响房价走势吗?》

发表于 2020-01-30 更新于 2020-02-06 阅读次数：
本文字数： 2.3k 阅读时长 ≈ 2 分钟

到底是为了解决什么样的问题，国家才提出了LPR（贷款基础利率）呢？——为了建设市场经济，向市场靠拢。

银行给个人或企业贷款，个人或企业偿还本金和利息；
计算利息的时候有利率 

利率 = 基准利率*(1+浮动)
基准利率由中央银行（政策）决定；
浮动的影响因素是多方面的，包括：央行、政策、银行、个人（抵押、资质）

房贷：4.9%（五年都没变过了）

首套：4.9%+(1+10%) = 5.39%
二套：4.9%+(1+30%) = 6.37%
政府调控的是前边的基准利率，调高了，每年还的钱就变多了。

为什么要引入LPR利率。因为目前的利率政策存在利率双轨制的问题。什么是利率双轨制呢？

简单来说，市场利率和政策利率是不一样的。
什么是市场利率：市场对于资金的需求程度就决定了利率，如果市场非常需要资金，那么利率自然而然就会升高。市场如果资金比较充裕不需要资金，利率就会下降。

市场利率是怎么出现的呢？
央行可以向商业银行做投放，投放资金，但不能随便投，不能印了钱就给商业银行。它需要通过一定的方式，比如最主要的方式是MLF（中期借贷便利，有人管它叫麻辣粉）。通过这种方式，央行把资金投放给商业银行。商业银行需要把自己的一些抵押物抵押给央行。然后在一段时间内，把本金和利息都还给央行。那么这种情况下，这MLF这个钱也不是白给的，你需要付利息的，所以说这个商行还得把一个利息加上这个本金再还给央行。于是，就存在一个MLF的利率。这个利率是央行决定的。央行可以说，我现在想要市场上钱多一些，于是它可以增加MLF的数额，同时降低MLF的利率，这样一来，商行就更愿意从央行里借钱。借完钱投放到市场，这市场上的钱不就多了吗？那，如果央行说，我今天不想让钱多了，怎么办？我就可以缩小MLF的数据，同时提高它的利率。那商行就不愿意借了，对不对？不愿借了之后，市场上钱不就少了吗？当然，有人说，那我央行随便一拍脑袋，我说我MLF的这个利率100%，行不行？不行，因为商行可以选择不借，你如果利率这么高，100%，那我就不管你借了，我管别的商行去借，不是一样吗？对不对？所以说，这是一种市场化的调控手段。虽然，他也是一种调控手段，但是它是一种市场化的调控手段，和那种政策手段是不一样的，那么这样就形成了一个利率，这个利率就称之为市场利率，它主要是由市场决定的。

但是还有一种利率，我们称之为政策利率。政策利率就是央行想定多少就定多少，不用去管市场的情况，这个政策利率主要就是指那个基准利率。基准利率加一定的浮动贷给个人和企业。所以这样就出现了一个问题，就是货币政策传导不畅。什么意思呢？比如说，这央行说我现在想要钱多一点，我就增加MLF的投放，让这个利率低一点，这样钱不就到商行了吗。但是它如果不调整这个基准利率的话，商行想贷给个人和企业的利率还是很很高。个人和企业就不从商行贷款。这个钱还是没办法到实体经济。商行拿这个钱又贷不出去，怎么办？所以，他们想方设法要把这个钱送出去，于是就想办法，比如房地产企业我多给你点，行不行，大型国企我多给你点，行不行。但是个人和小企业，一般就很难从这里边拿到钱了，对吧。

所以我们说那我们可以同时调整MLF和基准利率，那也是可以的，不过这样的话，就涉及到你又是用政策的手段来影响市场了。于是，我们说现在我们要建设市场经济，我们应该向市场靠拢，那怎么办，我们就应该取消政策利率。也就是说，把这个基准利率我不要了。我不要了之后，那么我参考什么样的标准来给个人和企业发放贷款呢？

这就是我们现在要做的LPR利率。
所以，为了解决利率双轨制的问题，国家就提出了现在所谓的LPR利率（贷款基础利率）。

LPR是什么？

LPR = MLF + 加点。
MLF由央行决定。
加点由商业银行决定（利润、风险），很多家银行一起决定LPR。

商业银行贷款给个人，现在的利率是，LPR+加点。
LPR的定价到底怎么定价，18家商业银行一起报价来决定，去掉最高和最低，剩下的16家加和取平均。这些银行代表不同的特色，服务不同的人群，能够代表整体的市场情况，由你们每个月来报价。

通过这样的改革，央行可以取消基准利率，每一次央行只需要调整MLF的利率，然后由商行决定LPR，这样就向这个市场经济更加靠拢了，于是就把两轨并作了一轨，就向这个市场利率靠拢。

这种情况下，房贷的利率是怎么算出来的？
房贷：利率 = LPR + 加点，比如：

首套： = 4.8% + 0 (政策加点)+ 0.5%（商业银行和个人协商的结果）= 5.3%
二套 = 4.8% + 0.6% (政策加点)+ 0.5%（商业银行和个人协商的结果）= 5.9%

房屋贷款利率有三部分组成：LPR利率（由市场决定），国家政策调控决定的，银行和个人协商的结果（哪家银行便宜用哪家）。

对于已经签订合同的贷款者来说，第一项是会变化的，但是第二项和第三项是不变的。

最后一部分是，利率的转换：

转换成LPR利率，或者固定利率。都需要锚定2019年12月20日。
首先，5.39% = 4.8% + ？ ；这个加点是多少了呢？是0.59%。从此以后，这个加点数就不变了，于是，未来很多年，你的利率就会这么算：

利率 = LPR + 0.59%

以下是，只看了一遍后，写的理解，有些不正确。
目前的市场化程度还不够高，调节一个事情，既需要市场利率的参与也需要政策利率的参与。

比如，央行想让钱更多的流向个人和中小企业。那么会先把钱给商业银行钱，商业银行给央行利息和本金，如果央行的利率过高，商业银行就可以选择不借钱。

商业银行拿到钱后，要给个人或者中小企业，这里的利率是基准利率，是政府调控的。如果这个利率过高，个人和中小企业就不会要。所以，政府为了让钱流入到个人或中小企业手中，需要通过政策降低这部分的利率。
