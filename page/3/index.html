听完预答辩后的收获

发表于 2020-01-09 更新于 2020-02-08
本文字数： 254 阅读时长 ≈ 1 分钟

如果用一句话总结今天的收获，那就是：在课题开展前，一定要把课题研究的意义，研究的框架理清楚了。不然，做得事情讲不圆一个故事，令人遗憾。

今天的预答辩，持续了5个多小时。临床专家，最关心的是背景和意义。到底对临床有没有价值，有没有意义，能不能用？其次，是描述的严谨性和逻辑性。博士生讲出来的话要有逻辑，有道理，不严谨是无法说服别人的。

在答辩的时候，不能照着稿子念，要富有情感，有感染力。

对我的提醒，
三叉神经痛，口腔癌，临床的现状到底是什么？
个例情况，要了解，深入临床，深入患者；
统计情况怎么样，普遍的困难是什么？

裤子在哪

发表于 2019-10-30 更新于 2020-01-30
本文字数： 383 阅读时长 ≈ 1 分钟

昨日与北京大学狼人杀王者某壮，进行乒乓球比赛。在于他进行了激烈的角逐后，比分进入了0:5，我暂时领先5局。在这5局中，某壮一直表示自己穿了牛仔裤，裤裆太紧，严重限制自己的水平发挥。我甚是唏嘘，遂提出合理建议：把裤子脱了，如何？某壮略加思索，面色绯红，表示有些不妥，等到0:7时，再脱不迟。我轻轻地点了点头。一眨眼，比分进行入了0:7。我挑衅地望着某壮，示意他即使脱了裤子，也未必能扳回一局。他受到了巨大的刺激，像头发疯的公牛跑到角落把裤子脱了。我清楚的知道，这一刻终将会被载入史册。毕竟，我把别人裤子打没了。轻装上阵的某壮似乎确实不一样，发球的力度和角度更加刁钻了，然而，比分并没有因为裤子没了被感动。又是一眨眼，比分进入0:11。从脱下裤子那一刻，看着岳某扬起的嘴角慢慢褪去，我有些自责，今天确实过分了。
