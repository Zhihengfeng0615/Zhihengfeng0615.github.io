读《如何阅读一本书》

发表于 2020-01-26 更新于 2020-02-08
本文字数： 278 阅读时长 ≈ 1 分钟

阅读的目标：为获得资讯而读，以及为求得理解而读。
总有一些书呆子读的太广，却读不通

读到5%的时候，我终于忍不住去查了查译者的背景，果然不是大陆人，翻译的语句读起来好绕口。

笔记-重点：
阅读的艺术就是要以适当的顺序提出适当的问题，关于一本书，你一定要提出四个主要的问题：

  1. 从整体来说，这本书到底在谈些什么？你一定要想办法找出这本书的主题，作者如何依次发展这个主题，如何逐步从核心主题分解出从属的关键议题来。
  2. 作者细节说了什么，怎么说的？
  3. 这本说说的有道理吗？是全部有道理，还是部分有道理？
  4. 这本书跟你有什么关系，如果这本书给了你一些资讯，你一定要问问这些资讯有什么意义！

听完预答辩后的收获

发表于 2020-01-09 更新于 2020-02-08
本文字数： 254 阅读时长 ≈ 1 分钟

如果用一句话总结今天的收获，那就是：在课题开展前，一定要把课题研究的意义，研究的框架理清楚了。不然，做得事情讲不圆一个故事，令人遗憾。

今天的预答辩，持续了5个多小时。临床专家，最关心的是背景和意义。到底对临床有没有价值，有没有意义，能不能用？其次，是描述的严谨性和逻辑性。博士生讲出来的话要有逻辑，有道理，不严谨是无法说服别人的。

在答辩的时候，不能照着稿子念，要富有情感，有感染力。

对我的提醒，
三叉神经痛，口腔癌，临床的现状到底是什么？
个例情况，要了解，深入临床，深入患者；
统计情况怎么样，普遍的困难是什么？
